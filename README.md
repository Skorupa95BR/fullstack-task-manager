# CRUD Node.js Project

Este projeto é um exemplo de aplicação CRUD (Create, Read, Update, Delete) utilizando Node.js.

## Tech Stack

**Frontend**
- React
- Tailwind

**Backend**
- Node.js
- Express

**Database**
- PostgreSQL (Neon)

**Deploy**
- Vercel
- Render

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure as variáveis de ambiente no arquivo `.env`.
3. Inicie o servidor:
   ```bash
   node server.js
   ```

## Estrutura
- `server.js`: Arquivo principal do servidor.
- `.env`: Variáveis de ambiente (não versionado).
- `package.json`: Dependências e scripts.

## Observações
- O arquivo `.env` está protegido pelo `.gitignore` e não será versionado.
- Certifique-se de criar o `.env` com as variáveis necessárias antes de rodar o projeto.
